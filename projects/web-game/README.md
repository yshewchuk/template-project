# Web Game

Browser-based Minesweeper built with TypeScript, React, and Vite.

## Stack

- **Language:** TypeScript
- **Framework:** React
- **Bundler:** Vite
- **Test Framework:** Vitest
- **Linter:** ESLint
- **Coverage:** v8/istanbul

## Development

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Run tests
npm test

# Run linter
npm run lint
```

## Directory Structure

```
web-game/
├── src/           # Application source code
├── tests/
│   ├── unit/          # Unit tests
│   ├── integration/   # Integration tests
│   └── e2e/           # End-to-end tests
└── docs/          # Project-specific documentation
```
